
# What is it?
____________________________________________________________________________________________________
A minimalist Minecraft resource pack that makes all armor labeled with 'inv' in its name invisible. Perfect for players seeking a clean look without visible armor, this pack lets you keep protection and functionality while maintaining an unarmored appearance. Ideal for role-players, content creators, and those looking to enhance immersion in Minecraft!

# Required?
____________________________________________________________________________________________________
## before 1.21.5:
- [CIT Resewn](https://modrinth.com/mod/cit-resewn)

- [Variants-CIT](https://modrinth.com/mod/variants-cit)
- [[ETF] Entity Texture Features](https://modrinth.com/mod/entitytexturefeatures)
- [[EMF] Entity Model Features](https://modrinth.com/mod/entity-model-features)
